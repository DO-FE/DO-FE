Dự án

ShopMates

Project này mình code bằng C# khi mới bắt đầu học code C# ASP.NET 6. Đây là project mình học từ TDEU

Jenkins

Jenkins này là được xây dựng các jobs tự động dự án Java và C# và Xây dựng cơ sở hạ tầng tự động bằng cách sử dụng Docker

Kubernetes

Đây là kubernest giúp quản lí các container và image dễ dàng. Xây dựng và triển khai ổn định và dể dàng. Sao chép cái token này để đăng nhập vào hệ thống: [Token_Kubernetes]

Jfrog Artifactory

Jfrog Artifactory này được sử dụng để lưu trữ registry docker và gói code khi nó được xây dựng. Sử dụng tài khoản này để vào: testing/Admin@123

Ceph with ansible

Đồ án này là luận văn tốt nghiệp của mình khi còn học ở FPT. Nó sẽ triễn khai quản lí các ổ cứng từ xa. Bây giờ không có server để triển khai nó. Có thể tải file tài liệu để xem chi tiết
